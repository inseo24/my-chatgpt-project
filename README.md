## ChatGPT API를 써보자

### 준비

- Node.js
- npm or yarn

### 설치


1. Clone the repository:

```bash
git clone git@github.com:inseo24/my-chatgpt-project.git
cd my-chatgpt-project
```

2. Install dependencies

```bash
npm install
```

or

```bash
yarn install
```

3. Add OpenAI API Key to .env.local file

```
OPENAI_API_KEY=your-api-key-here
```

4. Run the App

```
npm run dev
```

or 

```
yarn dev
```
